# Gemini Repository Guide

When this repository is used for packet capture analysis, prefer the `wireshark-traffic-analysis` project skill.

Canonical skill:

- `skills/wireshark-traffic-analysis/SKILL.md`

Mirrors for compatibility:

- `.github/skills/wireshark-traffic-analysis/SKILL.md`
- `.claude/skills/wireshark-traffic-analysis/SKILL.md`

Expected output should be structured, reproducible, and backed by packet-level evidence.
