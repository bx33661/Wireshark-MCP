# Wireshark MCP Docs Site

This directory contains the Astro Starlight documentation site for Wireshark MCP.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The site content lives in `src/content/docs/`. English pages are served from the root locale and Simplified Chinese pages live under `src/content/docs/zh-cn/`.
