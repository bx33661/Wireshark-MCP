## 环境要求

- Python 3.10 或更新版本。
- 已安装 Wireshark，并包含 `tshark`。
- 一个 MCP 客户端。

`tshark` 是唯一必需的 Wireshark CLI 依赖。`capinfos`、`mergecap`、`editcap`、`dumpcap`、`text2pcap` 等工具会被自动探测，存在时启用额外功能。

## 从 PyPI 安装

```sh
pip install wireshark-mcp
```

然后配置支持的 MCP 客户端：

```sh
wireshark-mcp install
```

安装后重启 AI 客户端。

<Tabs>
  <TabItem label="macOS">
    <Steps>

## 验证安装

```sh
wireshark-mcp doctor
```

当客户端无法启动服务、`tshark` 缺失，或实时抓包需要额外权限时，优先运行 `doctor`。

## 查看生成配置

JSON 类型 MCP 客户端：

```sh
wireshark-mcp config
```

Codex TOML：

```sh
wireshark-mcp config --format codex-toml
```

生成配置会使用当前 Python 解释器，并在需要时包含探测到的 Wireshark 工具绝对路径。

## 从源码安装

测试未发布改动时使用源码安装：

```sh
pip install git+https://github.com/bx33661/Wireshark-MCP.git
wireshark-mcp install
wireshark-mcp doctor
```

## 卸载

从支持的客户端中移除 MCP 服务配置：

```sh
wireshark-mcp uninstall
```
