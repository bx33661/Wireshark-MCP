## 快速开始

```sh
pip install wireshark-mcp
wireshark-mcp install
wireshark-mcp doctor
```

你需要 Python 3.10 或更新版本、带 `tshark` 的 Wireshark，以及 Claude Desktop、Claude Code、Cursor、VS Code、Codex CLI 等 MCP 客户端。

## 第一次调查

1. 用 `wireshark_open_file` 打开抓包。
2. 用 `wireshark_quick_analysis`、协议层级、端点和会话建立上下文。
3. 选择 playbook：triage、security、incident response、troubleshooting 或 CTF。
4. 用包详情、流跟踪、上下文包和字段提取确认线索。
5. 用证据和置信标签写结论。

## 证据优先

Wireshark MCP 面向可复现的数据包分析。重要结论应包含发现信号的工具调用、确认它的帧或流证据，以及清晰的置信标签：`confirmed`、`likely`、`possible` 或 `unresolved`。
