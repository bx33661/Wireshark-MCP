## 代理工作流

| 工具 | 用途 |
| --- | --- |
| `wireshark_quick_analysis` | 文件信息、协议分布、主要通信方、会话、主机名和异常摘要。 |
| `wireshark_security_audit` | 覆盖威胁情报、凭据、扫描、隧道、明文、专家信息和风险评分的安全审计。 |

## 入口工具

| 工具 | 用途 |
| --- | --- |
| `wireshark_open_file` | 打开抓包并激活上下文工具推荐。 |
| `wireshark_get_capabilities` | 列出可用工具和已探测到的 Wireshark suite 支持。 |

## 统计

| 工具 | 用途 |
| --- | --- |
| `wireshark_stats_protocol_hierarchy` | 理解协议组成和嵌套结构。 |
| `wireshark_stats_conversations` | 按包数、字节数和传输层优先排序主机对。 |
| `wireshark_stats_endpoints` | 建立主机清单并识别大流量通信方。 |
| `wireshark_stats_expert_info` | 发现错误、警告、重传、reset 和畸形包线索。 |
| `wireshark_stats_io_graph` | 查看流量随时间变化。 |
| `wireshark_stats_service_response_time` | 查看 HTTP、DNS、SMB 等响应时间统计。 |
| `wireshark_plot_traffic` | 生成 ASCII 流量柱状图。 |
| `wireshark_plot_protocols` | 生成 ASCII 协议层级树。 |

## 协议分析

| 工具 | 用途 |
| --- | --- |
| `wireshark_analyze_tcp_health` | 诊断重传、重复 ACK、零窗口和 RST。 |
| `wireshark_analyze_quic` | 提取 QUIC 版本、连接 ID、SNI 和 HTTP/3 帧。 |
| `wireshark_analyze_websocket` | 报告帧类型、载荷长度和掩码。 |
| `wireshark_analyze_mqtt` | 提取消息类型、主题、QoS 和客户端 ID。 |
| `wireshark_analyze_grpc` | 检查 HTTP/2 上的方法路径和消息长度。 |

## 提取

| 工具 | 用途 |
| --- | --- |
| `wireshark_extract_http_requests` | 提取 HTTP 方法、主机和 URI。 |
| `wireshark_extract_dns_queries` | 提取查询名和记录类型。 |
| `wireshark_extract_tls_handshakes` | 提取 TLS 版本、密码套件、SNI 和证书颁发者元数据。 |
| `wireshark_extract_smtp_emails` | 提取 SMTP 发件人、收件人和主题。 |
| `wireshark_extract_dhcp_info` | 提取租约、主机名、DNS 服务器和 DHCP 选项。 |
| `wireshark_extract_credentials` | 识别明文凭据暴露。 |
| `wireshark_export_objects` | 从支持协议中导出嵌入对象。 |
| `wireshark_list_ips` | 按方向列出唯一 IP。 |
