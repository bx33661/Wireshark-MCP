## Start Fast

```sh
pip install wireshark-mcp
wireshark-mcp install
wireshark-mcp doctor
```

You need Python 3.10 or newer, Wireshark with `tshark`, and an MCP-compatible client such as Claude Desktop, Claude Code, Cursor, VS Code, or Codex CLI.

## First Investigation

1. Open the capture with `wireshark_open_file`.
2. Build context with `wireshark_quick_analysis`, protocol hierarchy, endpoints, and conversations.
3. Pick a playbook: triage, security, incident response, troubleshooting, or CTF.
4. Confirm leads with packet details, stream follow-up, context packets, and field extraction.
5. Write findings with evidence and confidence labels.

## Evidence-first analysis

Wireshark MCP is designed for reproducible packet analysis. Important findings should include the tool call that found the signal, the frame or stream evidence that confirms it, and a clear confidence label: `confirmed`, `likely`, `possible`, or `unresolved`.
